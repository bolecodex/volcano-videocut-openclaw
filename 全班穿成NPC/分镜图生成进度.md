## 如何继续生成

使用 **novel-04-shots-to-images** skill，按场景逐批生成：

1. 选场景：`SC_01`（先生成剩余 14 张）→ `SC_02` → ... → `SC_05`
2. 每场景生成完成后，会写入对应 YAML 的 `image_url`、`image_status: completed`
3. 全部完成后运行：
   ```bash
   python .cursor/skills/novel-03-scenes-to-storyboard/scripts/generate_storyboard.py --project "全班穿成NPC/"
   open 全班穿成NPC/index.html
   ```

## 模型与参数

- **无角色**：`fal-ai/bytedance/seedream/v4.5/text-to-image`，仅 `prompt`
- **有角色**：`fal-ai/bytedance/seedream/v4.5/edit`，`prompt` + `image_urls`（来自 `全班穿成NPC_角色资产.yaml`）
- **图片尺寸**：从 `style.yaml` → `image_sizes.storyboard.preset` 读取（当前为 `portrait_16_9`）
