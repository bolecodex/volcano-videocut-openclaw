# 分镜划分规则

本规则用于在**单个场景内**划分镜头。

**执行前提**：novel-02-script-to-scenes 已完成场景切分，场景边界已确定。

## 核心原则

分镜的本质是**视觉叙事单元**。每个镜头应该能用**一张图片**准确表达该段落的核心内容。

> **关键理念**：分镜是按「画面」划分，场景是按「时空」划分。
> 场景切分已解决时空边界问题，分镜划分专注于画面切分。

### 重要约束

- **只在单场景内划分镜头**，不跨场景操作
- **不处理场景边界**，场景边界由 novel-02-script-to-scenes 决定
- **遵守场景元信息**，镜头的地点、时间、类型继承自所属场景

## 镜头容量硬性标准

### ⭐ 音频时长上限（第一指标）

> **最重要的指标**。每个镜头对应一张静态图/短视频（3-7秒）+配音。
> 音频时长必须匹配视频时长，超出会导致：1）画面与音频不同步 2）节奏拖沓 3）观众流失

**目标时长：3-7 秒/镜头**

**时长估算公式**：`估算时长(秒) = 总字符数 ÷ 4`（中文语速约 4 字/秒）

**字符计数规则**：
- 1 个汉字 = 1 字符
- 标点符号 = 0（不计入）
- 省略号（……）= 额外 +0.5 秒停顿
- 破折号（——）= 额外 +0.5 秒停顿

| 等级 | 估算时长 | 总字符数参考 | 处理方式 |
|------|----------|-------------|----------|
| ✅ 理想 | 3-5秒 | 12-20字 | 节奏最佳，直接使用 |
| ✅ 可用 | 5-7秒 | 20-28字 | 可接受 |
| ⚠️ 边界 | 7-8秒 | 28-32字 | 仅当单条长句无法自然断句时接受 |
| ❌ 必须拆分 | >8秒 | >32字 | 必须拆分，寻找断句点或分镜头 |

> **关键约束**：如果对话脚本中有单条台词 > 25 字，应在 novel-02-script-to-scenes 阶段就进行断句处理。
> 分镜阶段遇到长句（>28字），应将该句独占一个镜头，不再叠加其他台词。

**实际案例（3-7s 标准）**：
```
❌ 超标示范：SC_02_001（4条台词，123字，~31s → 应是4-5个镜头而非1个）
   - "嫁给陈屠的第三个月...覆盖了" (41字, 10s) → 独占1镜头，建议上游断句
   - "西市的街坊们...窃窃私语" (36字, 9s) → 独占1镜头，建议上游断句
   - "苏家的大小姐...杀猪的？" (21字, 5s) → ✅ 可独立
   - "嘘——家道中落...卖进窑子强" (25字, 6s) → ✅ 可独立

✅ 正确拆分（如果上游已断句）：
   镜头A | "嫁给陈屠的第三个月，我身上从苏府带出来的最后一缕檀香，" (25字, ~6s) ✅
   镜头B | "终于被那猪下水的味道彻底覆盖了。" (15字, ~4s) ✅
   镜头C | "西市的街坊们看我的眼神，惋惜里掺着看好戏的幸灾乐祸，" (25字, ~6s) ✅
   镜头D | "总有人窃窃私语，" + 街坊甲台词 (9+21=30字, ~8s) ⚠️ 边界
   镜头E | 街坊乙台词 (25字, ~6s) ✅
```

### 对话条数上限（辅助指标）

条数作为**辅助检查**，在 3-7s 时长约束下，条数自然会被限制。

| 等级 | 对话条数 | 处理方式 |
|------|----------|----------|
| ✅ 理想 | 1-2条 | 典型分镜：1条中等句 或 2条短句 |
| ⚠️ 可接受 | 3条 | 仅当全部为极短句（每条≤8字）时可接受 |
| ❌ 必须拆分 | 4条以上 | 无论多短都必须拆分 |

> **核心逻辑**：3-7s 意味着最多 28 字。
> 1 条 25 字的句子就占满了一个镜头，不可能再塞第二条。
> 只有全部是极短句（"吃。"+"他转身。"+"我愣住了。"）才可能 3 条在一起。

### 单条台词长度处理

| 单条字符数 | 处理方式 | 说明 |
|-----------|---------|------|
| ≤ 10字 | 与相邻短句合并 | "吃。"(1字) + "他转身去切肉了。"(7字) = 8字 ✅ |
| 11-25字 | 理想长度 | 独占镜头或搭配1条极短句 |
| 26-28字 | 独占镜头 | 不再搭配任何其他台词 |
| 29-35字 | 独占镜头+标记 | 标记为"上游需断句"，临时接受 |
| > 35字 | ⚠️ 上游问题 | 此句应在 novel-02-script-to-scenes 阶段断句 |

> **重要**：分镜阶段遇到 >28 字的台词，是**上游 novel-02-script-to-scenes 的问题**。
> 分镜阶段的临时处理：让该长句独占镜头，不叠加其他台词。
> 根本解决：在 novel-02-script-to-scenes 阶段就将长句在自然断点处拆分。

```
示例：如何在分镜阶段处理上游长句

旁白："嫁给陈屠的第三个月，我身上从苏府带出来的最后一缕檀香，
       终于被那猪下水的味道彻底覆盖了。" (41字, ~10s)
→ 独占1个镜头（超标但无法在分镜层面拆分，标记需上游修复）

理想的上游断句（novel-02-script-to-scenes 层面）：
→ "嫁给陈屠的第三个月，我身上从苏府带出来的最后一缕檀香，" (25字, ~6s) ✅
→ "终于被那猪下水的味道彻底覆盖了。" (15字, ~4s) ✅
→ 各占1个镜头，完美匹配3-7s目标
```

### 时间跨度检测（强制拆分）

**同一镜头内禁止出现的时间跳跃**：

| 时间跳跃类型 | 触发词示例 | 处理 |
|-------------|-----------|------|
| 日夜交替 | "天亮了"、"入夜后" | 必须拆分 |
| 隔天 | "第二天"、"次日清晨" | 必须拆分 |
| 餐点切换 | "晚饭时...夜里..." | 必须拆分 |
| 明确时段 | "子时→天色泛白" | 必须拆分 |

### 旁白时空跳跃检测（强制拆分）⚠️ 新增

> **关键规则**：即使是连续旁白，如果描述了不同时空的事件，也必须拆分镜头！
> 旁白不等于"可以塞在一个镜头里"，旁白也是视觉叙事的一部分。

**旁白时空跳跃触发词**：

| 跳跃类型 | 触发词示例 | 处理 |
|----------|-----------|------|
| 地点变化 | "被拖出闺房"、"扔进人市" | 必须拆分 |
| 人物命运切换 | "我爹...我娘..."、"他死后...她也..." | 必须拆分 |
| 事件切换 | "抄家那日...下了诏狱...悬了梁..." | 每个事件一镜头 |
| 视角切换 | "他们看我...我只当..." | 考虑拆分 |

**错误示范**：
```
❌ SC_03 苏府抄家（12句旁白塞1个镜头）：
   旁白：苏家被抄那日，大雪封了京城。           ← 时空1：苏府
   旁白：我爹...递上折子...批了四个字...        
   旁白：人，就直接下了诏狱。                   ← 时空2：诏狱
   旁白：我娘...当晚便悬了梁...                 ← 时空3：母亲房间
   旁白：我被官差从闺房里拖出来...扔进了人市。  ← 时空4：人市
   旁白：那些公子哥儿们...轻佻的目光...         ← 情绪转折
   旁白：就在我冻得快要失去意识的时候...        ← 人物入场
   → 虽然都是旁白，但跨越4个时空+2个转折，应拆为5个镜头！
```

**正确拆分**：
```
✅ SC_03_001 | 苏府抄家（2条旁白）
   画面：大雪中苏府被围，官差冲入
   
✅ SC_03_002 | 诏狱酷刑（2条旁白）
   画面：诏狱阴暗，暗示父亲受刑

✅ SC_03_003 | 母亲自缢（1条旁白）
   画面：白绫垂下，母亲房间

✅ SC_03_004 | 人市受辱（4条旁白）
   画面：苏晚跪在人市木台，公子哥围观

✅ SC_03_005 | 陈屠出现（3条旁白）
   画面：高大黑影遮挡寒风
```

### 动作密度检测（建议拆分）

当一个镜头内有 **3个以上独立动作高潮** 时，应考虑拆分：

```
❌ SHOT_17 动作堆叠：
   1. 陈屠劈案板（动作高潮1）
   2. 斩骨刀甩出（动作高潮2）
   3. 白马受惊扬蹄（动作高潮3）
   4. 沈清河摔马（动作高潮4）
   5. 陈屠说"滚"（台词高潮）
   → 建议拆为 2 个镜头：劈刀+甩刀 / 摔马+护妻
```

## 触发新镜头的条件

> **注意**：场景切换和时间跳跃已在 Phase A 处理，这里只处理**场景内**的镜头划分。

### 1. 人物重大出入场（建议）

当重要人物出入场时，考虑划分新镜头：

```
✅ 新镜头触发：
- 新角色登场（首次亮相）
- 多个角色同时进入画面
- 角色退场后其他角色特写

示例：
旁白：陈屠不知何时已经站在了案板后面。  ← 考虑新镜头：陈屠入场
```

### 3. 情绪剧烈转折（建议）

当情绪发生180度转变时，考虑划分新镜头：

```
✅ 情绪转折标志：
- 从平静到激动
- 从悲伤到希望
- 从恐惧到安心
- 重大发现、震惊时刻

示例：
旁白：我心头一跳。  ← 考虑新镜头：情绪从平静转为紧张
```

### 5. 动作高潮（建议）

重要的动作场面应独立成镜：

```
✅ 动作高潮：
- 打斗场面
- 追逐场面
- 关键动作（劈刀、摔倒等）

示例：
旁白："砰！咔嚓——！！！"  ← 新镜头：陈屠劈案板的戏剧动作
```

## 不触发新镜头的情况

> **重要前提**：以下"不拆分"规则仅适用于 **同一时空、同一画面焦点** 的情况。
> 如果旁白/对话涉及时空跳跃，即使是连续的也必须拆分！

### 1. 短句合并（总字符数 ≤ 28，同一时空/画面）

极短句（≤10 字）不应单独成镜，应与相邻短句合并：

```
✅ 合并成功（共2条，16字，~4s）：
陈屠：吃。(1字)
旁白：他只说这一个字，转身又去切肉去了。(15字)
← 合并后 16 字 ≈ 4s ✅

✅ 合并成功（共2条，21字，~5s）：
旁白：我小口啃着饼。(7字)
旁白：芝麻粒在齿间迸开，焦香四溢。(14字)
← 合并后 21 字 ≈ 5s ✅

✅ 合并成功（共3条，16字，~4s）：
苏晚：你去哪？(3字)
旁白：我哑着嗓子问。(7字)
陈屠：外面有长凳。我守夜。(9字)
← 合并后 19 字 ≈ 5s ✅ （但注意3条是上限）

❌ 合并失败（超过 28 字）：
旁白：是我的夫君，陈屠，(9字)
旁白：每当那些闲言碎语飘进来，他剁骨头的声音就格外响。(23字)
← 合并后 32 字 ≈ 8s ❌ 超标，必须分别成镜
```

### 2. 不合并的情况

以下情况**即使字符数允许也不合并**：

```
❌ 时空跳跃不合并：
旁白：我爹下了诏狱。(7字)
旁白：我娘悬了梁。(6字)
← 虽然只有 13 字，但跨时空，必须各自成镜！

❌ 画面焦点变化不合并：
旁白：我只当听不见，低头舀起滚烫的水。(16字) ← 画面：苏晚
旁白：案板那边，"砰"一声闷响。(11字) ← 画面：案板/陈屠
← 虽然只有 27 字，但画面焦点从苏晚切到陈屠，应分镜

❌ 说话人交替不合并（除极短句）：
苏晚：当家的，张侍郎……没了。(11字)
陈屠：嗯。(1字)
← 虽然只有 12 字，但这是一组试探对话，
   视觉上需要切镜头表现两人表情，应分镜
```

## 镜头密度参考

> **适用媒介**：漫画分镜、动画分镜、有声漫画、短剧分镜
> 特点：每镜头对应一张静态图+配音，需要较高的镜头密度

| 内容类型 | 每10条台词 | 特点 |
|----------|------------|------|
| 纯对话场景（短句为主） | 3-4个镜头 | 表情/反应切换 |
| 叙事旁白场景（长句为主） | 5-7个镜头 | 长旁白需更多镜头承载 |
| 动作场景 | 5-6个镜头 | 动作分解 |
| 闪回/蒙太奇 | 5-8个镜头 | 快切节奏 |

**经验公式（以字符数为准）**：
```
预估镜头数 ≈ 总字符数 ÷ 20
```
（20 字 ≈ 5 秒，取 3-7 秒范围的中间值）

**短句合并规则**：
- 极短句（≤10 字）不应单独成镜，需与相邻短句合并
- 合并后总字符数仍须 ≤ 28 字
- 合并优先级：同一说话人 > 连续动作 > 相邻旁白

**示例**：
```
SC_02 西市日常（如上游已断句）：约 20 条台词，~327 字
  预估：327 ÷ 20 = ~16 镜头（短句合并后约 12-14 镜头）

SC_01 开篇悬念：7 条台词，~90 字
  预估：90 ÷ 20 = ~5 镜头

合并示例：
  ✅ "吃。"(1字) + "他只说这一个字，转身又去切肉去了。"(15字) = 16字, ~4s
  ✅ "我小口啃着饼。"(7字) + "芝麻粒在齿间迸开，焦香四溢。"(14字) = 21字, ~5s
  ❌ "是我的夫君，陈屠，"(9字) + "每当那些闲言碎语..."(23字) = 32字, ~8s → 超标
```

**对比参考**：
| 媒介 | 公式 | 原因 |
|------|------|------|
| 电影 | ÷8 | 镜头可动，画面承载信息量大 |
| 漫画/动画 | MAX(条数÷3, 字数÷40) | 静态图+配音，需严格控制时长 |
| 有声小说 | ÷10+ | 纯音频，无画面限制 |

## 拆分示例

### 示例1：时间跨度过长的镜头

**原始问题**（SHOT_08，18条对话）：
```
旁白：直到那晚，瓢泼大雨。              ← 夜晚开始
旁白：我被一声嘶吼声惊醒。
旁白：陈屠不知何时进了屋...
陈屠：……张……德坤……
...（噩梦内容）...
旁白：我缩在床角，抱着膝盖，睁着眼，直到天色泛白。  ← 时间跳跃
旁白：雨停了。陈屠醒来...
陈屠：吵到你了？
旁白：早饭时，我碗里多了个剥好的水煮蛋。  ← 又一次时间跳跃
```

**正确拆分**：
```
SHOT_08A | 雨夜惊梦
- 对话条数：10条
- 时间：雨夜
- 画面：陈屠躺在地上挣扎，苏晚惊恐缩在床角

SHOT_08B | 雨后清晨
- 对话条数：8条
- 时间：天亮后、早饭时
- 画面：陈屠递水煮蛋，苏晚若有所思
```

### 示例2：情节段落过长的镜头

**原始问题**（SHOT_19，41条对话）：
```
内容涵盖4个不同的情节单元：
1. 苏晚瘫坐，陈屠抱起照顾
2. 坦白"我知道你的秘密"
3. 陈屠崩溃倾诉过去
4. 夫妻结盟
```

**正确拆分**：
```
SHOT_19A | 劫后照顾
- 对话条数：8条
- 画面：陈屠将苏晚抱到床边，递热帕子

SHOT_19B | 坦白秘密
- 对话条数：12条
- 画面：苏晚说"我知道你在梦里做的事"，陈屠惊恐

SHOT_19C | 往事倾诉
- 对话条数：10条
- 画面：陈屠崩溃抱住苏晚，断续倾诉

SHOT_19D | 夫妻盟约
- 对话条数：11条
- 画面：苏晚递出书和本子，庄严立约
```

### 示例3：动作过多的镜头

**原始问题**（SHOT_17，23条对话，5个动作高潮）：
```
1. 陈屠劈案板
2. 斩骨刀甩出
3. 白马受惊
4. 沈清河摔马
5. 陈屠说"滚"
```

**正确拆分**：
```
SHOT_17A | 陈屠震慑
- 对话条数：12条
- 画面：陈屠举刀劈下，案板裂开，刀光凌厉

SHOT_17B | 前夫落马
- 对话条数：11条
- 画面：白马扬蹄，沈清河摔落，狼狈逃窜
```

## 特殊处理

> **注意**：梦境、闪回、蒙太奇的场景边界已在 Phase A 确定。
> 这里只处理这些**特殊类型场景内部**的镜头划分。

### 1. 梦境场景内的分镜

梦境场景通常需要较高的镜头密度：

```
场景元信息已标注：type: dream
镜头划分建议：
- 梦境入口/环境建立：1个镜头
- 核心动作/冲突：2-3个镜头
- 梦境退出/醒来：1个镜头
```

### 2. 蒙太奇场景内的分镜

蒙太奇场景的镜头数由场景元信息中的 `estimated_shots` 预估：

```
场景元信息已标注：
- type: montage
- estimated_shots: 6-8

按预估镜头数划分快切画面，每个镜头可能只有1-2条旁白
```

### 3. 闪回场景内的分镜

闪回场景的分镜方式与现实场景相同，但需要继承场景的视觉标记：

```
场景元信息已标注：
- type: flashback
- time_reference: 苏家被抄那日

所有镜头的 prompt 需包含闪回视觉标记（如暗角、褪色等）
```

## 分镜自检清单

完成划分后，逐项检查每个镜头：

### ⭐ 时长检查（最重要，第一优先级）
- [ ] 每个镜头的总字符数 ≤ 28？（目标 3-7 秒）
- [ ] 无镜头总字符数 > 32？（超过必须拆分）
- [ ] 单条台词字符数 ≤ 28？（超长需标记"上游需断句"）
- [ ] 极短句（≤10字）已与相邻句合并？（不独占镜头）
- [ ] 镜头间时长差异不超过 2 倍？（目标：3-7s 均匀）
- [ ] 无台词在多个镜头间重复出现？

### 条数检查（辅助）
- [ ] 对话条数 ≤ 2？（理想状态）
- [ ] 对话条数 ≤ 3？（仅当全部极短句时可接受）
- [ ] 无镜头对话条数 ≥ 4？

### 时空一致性检查（关键）
- [ ] 每个镜头内的旁白描述**同一时空**？
- [ ] 旁白中的"地点变化"已拆分为新镜头？
- [ ] 旁白中的"人物命运切换"（A死了...B也...）已拆分？
- [ ] 镜头的 scene_ref 正确引用了场景ID？
- [ ] 特殊类型场景（梦境/闪回）的镜头继承了视觉标记？

### 动作检查
- [ ] 动作高潮点 ≤ 2 个？（超过考虑拆分）
- [ ] 无多个独立的戏剧性动作被压缩？

### 画面检查（核心）
- [ ] 能用**一张图**清晰表达该镜头的核心内容？
- [ ] 该镜头的画面焦点是**明确单一**的？
- [ ] 估算音频时长 ≤ 7秒？（`总字符数 ÷ 4 ≤ 7`）

### 整体检查
- [ ] 各镜头时长差异不超过 2 倍？（最长 vs 最短）
- [ ] 单场景镜头数 ≈ 总字符数 ÷ 20？
- [ ] 蒙太奇场景的镜头数符合预估？

### 常见错误自查
- [ ] 有没有"连续旁白塞一个镜头"但实际跨多个时空？
- [ ] 有没有"闪回场景只有1个镜头"但叙述了多个事件？
- [ ] 有没有极短句（≤10字）单独成镜浪费了一个画面？
- [ ] 有没有单条长句（>28字）未标记"上游需断句"？
- [ ] 有没有同一台词在多个镜头中重复？
