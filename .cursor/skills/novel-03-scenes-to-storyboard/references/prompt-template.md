# 配图提示词模板

本模板用于为每个镜头生成配图提示词。

**执行前提**：novel-02-script-to-scenes 完成场景切分，分镜划分已完成。

## 提示词结构

```
[风格锚点] + [场景描述] + [角色描述@ID] + [动作/表情] + [构图/镜头] + [技术参数]
```

## 风格锚点

风格锚点放在提示词开头，确保全剧视觉一致性：

```

真人写实高清，超细节刻画，古风人像写实，光影细腻，氛围感拉满，服饰纹理清晰，面部神态精准，动态自然，暖黄色的光线，背景虚化氛围感强

# 其他风格参考
写实古装：电影质感，8K超清，古装剧剧照风格，自然光影
水墨风格：水墨画风格，留白意境，淡雅色调
赛博朋克：霓虹光影，雨夜氛围，高对比度
```

## 构图/镜头模板

### 景别

```
特写镜头，面部占据画面80%
中景镜头，人物膝盖以上
全景镜头，人物全身可见，环境占比40%
远景镜头，人物在画面中较小，环境为主
```

### 构图

```
居中构图，人物位于画面中央
三分构图，人物位于画面左/右三分之一处
对角线构图，人物沿对角线分布
框架构图，通过门窗等框住主体
```

### 视角

```
平视视角，与人物视线平齐
俯视视角，从上向下俯瞰
仰视视角，从下向上仰望
主观视角，模拟角色视角
```

## 常见问题

### Q: 角色在画面中太小怎么办？
A: 使用中景或特写，减少环境描述的篇幅

### Q: 多人场景如何处理？
A: 明确主次关系，主角描述详细，配角简略提及

### Q: 动态场景如何表现？
A: 添加"动态模糊"、"速度线"、"定格瞬间"等关键词
