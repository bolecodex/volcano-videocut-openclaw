```
[角色描述...]，
```

```
[角色描述...]，
{shot_type}镜头，迷幻氛围，恐怖神秘，高清细节
```

### 蒙太奇 (montage)

## 镜头类型对应构图

### 远景/全景

- 展示完整场景
- 人物较小，环境为主
- 建立空间感

```
远景镜头，展示完整{location}，{环境描述}，
人物位于画面{位置}，渺小身影，
大气透视，层次分明，高清细节
```

### 中景

- 人物腰部以上
- 表情和动作都能看清
- 最常用的镜头

```
中景镜头，{角色}位于画面中心，
动作清晰可见，表情生动，
{背景描述}略带景深，高清细节
```

### 近景

- 人物胸部以上
- 强调表情

```
近景镜头，聚焦{角色}，胸部以上，
神情{emotion}，表情细腻，
背景虚化，突出人物，高清细节
```

### 特写

- 面部或特定物体
- 最大程度强调细节

```
特写镜头，极致聚焦{焦点}，
{细节描述}，
浅景深，背景完全虚化，写实质感，高清细节
```
