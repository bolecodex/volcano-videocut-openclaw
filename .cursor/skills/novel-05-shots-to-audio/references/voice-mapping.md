## 语气词标签（speech-2.8 支持）

在文本中插入以下标签可增强情感表达：

| 标签 | 效果 |
|------|------|
| `(laughs)` | 笑声 |
| `(sighs)` | 叹气 |
| `(crying)` | 哭泣 |
| `(angry)` | 愤怒语气 |
| `(trembling voice)` | 颤抖的声音 |
| `(whispers)` | 低语/耳语 |
| `(shouts)` | 大喊 |

**使用示例**：

```
原文：夫君，今晚，替我多砍几刀。
情绪：颤抖、决绝

处理后：(trembling voice) 夫君，今晚，替我多砍几刀。
```

## 停顿控制

使用 `<#x#>` 标签控制停顿（x 为秒数）：

```
我那夫君，<#0.5#>有个秘密。
```

适用场景：
- 悬念停顿：`有个秘密<#1#>——`
- 语气停顿：`夫君，<#0.3#>今晚`
- 段落停顿：`...暴毙。<#1.5#>直到那晚...`
