# 剧本转对话格式规则详解

## 1. 输出结构规范

### 1.1 整体结构

```
[音色映射表]

[空行]

[对话内容]
```

### 1.2 音色映射表格式

```
旁白：[音色描述]
[人物1]：[音色描述]
[人物2]：[音色描述]
...
```

### 1.3 对话内容格式

```
旁白：[叙述内容]
[人物]：[对话内容]
旁白：[叙述内容]
[人物]：[对话内容]
...
```

## 2. 说话者识别规则

### 2.1 明确标注类型

| 标注方式 | 示例 | 说话者 |
|----------|------|--------|
| XX说 | 张三说："..." | 张三 |
| XX道 | 李四道："..." | 李四 |
| XX答道 | 老者答道："..." | 老者 |
| XX问 | 她问："..." | 她（需上下文确定） |
| XX笑道 | 师傅笑道："..." | 师傅 |
| XX怒道 | 他怒道："..." | 他（需上下文确定） |
| XX冷声道 | 陈屠冷声道："..." | 陈屠 |
| XX低声道 | 苏晚低声道："..." | 苏晚 |

### 2.2 上下文推断规则

当对话前没有明确说话者时：

1. **相邻原则**：对话通常属于最近提到的人物
2. **对话轮次**：连续对话通常是两人轮流
3. **内容分析**：根据对话内容判断（如称呼、语气）
4. **第一人称**：剧本中"我"的对话通常是主角

### 2.3 特殊情况处理

| 情况 | 处理方式 | 示例 |
|------|----------|------|
| 心理活动 | 保留为旁白或标注"XX内心" | 我心想：这下完了 → 旁白：我心想，这下完了 |
| 画外音 | 标注为"旁白"或"画外音" | （画外音）很久以前... → 旁白：很久以前... |
| 多人齐声 | 标注"众人"或列出所有人 | 众人齐声道 → 众人：... |
| 自言自语 | 标注角色名 | 他自言自语 → 张三：... |

## 4. 格式化输出规则

### 4.1 标点符号处理

| 原文 | 输出 | 说明 |
|------|------|------|
| "你好" | 你好 | 去掉引号 |
| 「你好」 | 你好 | 去掉直角引号 |
| '你好' | 你好 | 去掉单引号 |
| 他说："你好。" | 你好。 | 保留句末标点 |
| "真的？" | 真的？ | 保留问号 |
| "什么！" | 什么！ | 保留感叹号 |
| "嗯......" | 嗯...... | 保留省略号 |

### 4.2 段落处理

1. **保持原文段落结构**
2. **⭐ 每行一个说话人**：不同音色的台词**绝不合并**到同一行
3. **⭐ 长句必须断句**：单条台词超过 **15 个中文字符**时，必须在自然断句点（逗号、句号、分号）处拆分为多条
4. **⭐ 拆分不能太碎**：拆分后每段应至少 **5 个中文字符**，避免出现"夫君，""别哭，"这样的碎片行

### 4.3 台词长度控制（重要！）

> **核心原则**：每条台词控制在 **5-15 个中文字符**区间。
> 太长（>15 字）导致下游分镜无法合理分配时长；
> 太短（<5 字）导致配音碎片化、节奏破碎。
> 必须在本阶段控制好长度。

**长度标准**：

| 台词长度（中文字符数） | 处理 |
|------------------------|------|
| ≤ 15 字 | ✅ 直接输出 |
| 16-20 字 | ⚠️ 尽量在逗号处断句 |
| > 20 字 | ❌ 必须在逗号/句号处断句 |

**拆分下限**：

| 拆分后单段长度 | 处理 |
|----------------|------|
| ≥ 5 字 | ✅ 合格 |
| < 5 字（完整短句） | ✅ 例外允许（如"闭嘴。""来者何人？""走！"等本身就是完整语句） |
| < 5 字（碎片） | ❌ 不允许，应与相邻句段合并 |

**字符计数说明**：
- 只计算**中文字符**（汉字），不计标点符号、空格、英文字母
- 例："嫁给陈屠的第三个月" = 8 个中文字符 ✅
- 例："夫君今晚替我多砍几刀" = 10 个中文字符 ✅
- 例："一本弹劾丞相魏忠贤贪墨河工款的折子递上去" = 18 个中文字符 ❌ 需拆分

**断句规则**：
1. 在**逗号、句号、分号**等自然断句点拆分
2. 拆分后每段应有**独立语义**（不是半句话）
3. 拆分后每段应至少 **5 个中文字符**（完整短句除外）
4. 如果逗号处拆分会产生 < 5 字的碎片，应与前后句段合并
5. 拆分后的每条仍保持同一说话人标注

**断句示例**：

```
❌ 原文（20个中文字符，需拆分）：
旁白：嫁给陈屠的第三个月，我终于习惯了猪下水的味道。

✅ 断句后：
旁白：嫁给陈屠的第三个月，
旁白：我终于习惯了猪下水的味道。

---

❌ 原文（25个中文字符，需拆分）：
旁白：他一声不吭地走到隔壁烧饼铺，买了一个芝麻烧饼塞我手里。

✅ 断句后：
旁白：他一声不吭地走到隔壁烧饼铺，
旁白：买了一个芝麻烧饼塞我手里。

---

✅ 原文（10个中文字符，无需拆分）：
苏晚：夫君，今晚替我多砍几刀。

✅ 直接输出，不拆分。

---

❌ 原文（18个中文字符，需拆分）：
旁白：我爹，刑部侍郎苏蔺，一生刚正不阿。

✅ 断句后：
旁白：我爹，刑部侍郎苏蔺，
旁白：一生刚正不阿。

（注意："我爹，" 仅 2 字，不可单独成行，须与后面合并）

---

❌ 过度拆分（每段太短太碎）：
旁白：我爹，
旁白：刑部侍郎苏蔺，
旁白：一生刚正不阿。

✅ 合理拆分（每段 5-15 字）：
旁白：我爹，刑部侍郎苏蔺，
旁白：一生刚正不阿。
```

### 4.3 需要过滤的内容

| 类型 | 示例 | 处理 |
|------|------|------|
| 章节序号 | 01、02、第一章 | 过滤，不输出 |
| 纯标点行 | ——、...... | 过滤，不输出 |
| 空行 | （空） | 保留用于段落分隔 |
| 尾声/结语 | 尾声、完 | 过滤，不输出 |

## 5. 完整示例

### 示例1：古风悬疑

**输入（剧本原文）**：
```
01

大唐，长安。

夜色如墨，一道黑影掠过屋檐。

"来者何人？"守卫喝道。

黑衣人冷笑一声："阎王派我来取你性命。"

剑光一闪，守卫倒地不起。

02

清晨，狄公踏入现场。

"大人，死者是皇宫侍卫，名叫李忠。"元芳躬身禀报。

狄公蹲下身子，仔细查看尸体："致命伤在咽喉，一击毙命，凶手剑术精湛。"

"元芳，你怎么看？"

"大人，此事必有蹊跷！"
```

**输出（格式化后）**：
```
旁白：沉稳磁性男声
守卫：粗犷警觉男声
黑衣人：阴冷狠辣男声
狄公：沉稳睿智男声
元芳：恭敬干练男声

旁白：大唐，长安。
旁白：夜色如墨，一道黑影掠过屋檐。

黑衣人：阎王派我来取你性命。
```

## 6. 常见错误与修正

### 6.1 格式错误

| 错误示例 | 正确示例 | 问题 |
|----------|----------|------|
| `旁白："他说……"` | `旁白：他说……` | 旁白内容不加引号 |
| `张三:你好` | `张三：你好` | 使用中文冒号 |
| `01` | （不输出） | 章节序号需过滤 |
| `张三说："你好"` | `张三：你好` | 去掉引号和说话动词 |

### 6.2 识别错误

| 错误 | 正确 | 问题 |
|------|------|------|
| 把心理活动当对话 | 保留为旁白 | 内心独白不是实际对话 |
| 漏掉隐含说话者 | 根据上下文推断 | 需仔细分析语境 |
| 混淆相似角色 | 通过对话内容区分 | 注意称呼和语气 |

## 7. 质量检查清单

- [ ] 是否过滤了所有章节序号（01、02、第一章等）？
- [ ] 音色映射表是否包含所有出场人物？
- [ ] 音色描述是否符合角色性格？
- [ ] 所有对话是否正确识别了说话者？
- [ ] 旁白和对话的划分是否准确？
- [ ] 是否保持了原文的完整性（未删减内容）？
- [ ] 格式是否统一（中文冒号、去引号）？
- [ ] 段落分隔是否合理？
- [ ] 是否有遗漏的对话？
- [ ] 多人场景的说话者是否清晰？
- [ ] ⭐ 是否有单条台词超过 15 个中文字符未断句？
- [ ] ⭐ 断句后是否有碎片行（< 5 个中文字符且不是完整短句）？
- [ ] ⭐ 断句后的每条是否有独立语义（不是半句话）？
- [ ] ⭐ 每行是否只有一个说话人（不同音色不合并）？
