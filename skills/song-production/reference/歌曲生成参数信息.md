## 歌曲生成请求参数（v4.3） <a id="song-params-v4-3"></a>

| 参数              | 类型   | 必填 | 说明                                                                                                                                                                                                                         |
| ----------------- | ------ | ---- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Lyrics            | String | 否   | 可选参数：歌词内容，中文支持 5 到 700 个字符，英文支持 5-2000 个字符。支持通过歌词精准生成，与 Prompt 二选一，优先级高于 Prompt                                                                                             |
| Prompt            | String | 否   | 可选参数：提示词内容，仅支持中文，5 到 700 个字符。支持通过提示词生成歌曲，与 Lyrics 二选一                                                                                                                               |
| ModelVersion      | String | 否   | 固定为 "v4.3"。如不传值，则默认使用 v4.3                                                                                                                                                                                    |
| Genre             | String | 否   | 可选参数：生成歌曲的主曲风，如果不传或者传入空字符串，服务会基于 text 内容提取或预测 genre。v4.3 支持最多 3 个主曲风，用逗号分隔。在传入 Lyrics 且没有指定也没有传入 Prompt 时，默认为 Pop。可选值见 [Genre / GenreExtra 可选值](#genre-options-v4-3) |
| Mood              | String | 否   | 生成歌曲的情绪，如果不传或者传入空字符串，服务会基于 text 内容提取或预测 mood。v4.3 支持最多 2 个情绪词，用逗号分隔。可选值见 [Mood 可选值](#mood-options-v4-3)                                                            |
| Gender            | String | 否   | 生成歌曲的 Gender，如果不传或者传入空字符串，服务会基于 text 内容提取或预测 gender。可选参数：["Female", "Male"]                                                                                                          |
| Timbre            | String | 否   | 生成歌曲的音色，如果不传或者传入空字符串，服务会基于 text 内容提取或预测 timbre。v4.3 支持最多 3 个音色，用逗号分隔。可选值见 [Timbre 可选值](#timbre-options-v4-3)                                                       |
| Duration          | Int    | 否   | 可选参数：生成指定音频时长，单位：s，生效范围 [30，240]                                                                                                                                                                     |
| CallbackURL       | String | 否   | 可选参数：配置回调地址后，会在任务完成后，通过 http 请求回调该地址，回调内容为 json 格式                                                                                                                                 |
| SkipCopyCheck     | Bool   | 否   | 可选参数：歌词版权检测开关。ture（关闭）；false（开启）                                                                                                                                                                     |
| TosBucket         | String | 否   | 可选参数：用户自定义 tos 存储桶名称                                                                                                                                                                                         |
| ImplicitWaterMark | map    | 否   | 可选参数：隐私水印，子字段见下文                                                                                                                                                                                             |
| GenreExtra        | String | 否   | 4.3 参数：次级曲风，最多支持 2 个。可选值见 [Genre / GenreExtra 可选值](#genre-options-v4-3)                                                                                                                                |
| Key               | String | 否   | 新增 v4.3 参数，音乐调性。可选值：["A", "A#", "B", "C", "C#", "Cb", "D", "D#", "E", "F", "F#", "G", "Ab"]                                                                             |
| Kmode             | String | 否   | 4.3 参数：音乐调式。可选值见 [Kmode 可选值](#kmode-options-v4-3)                                                                                                                                                           |
| Tempo             | String | 否   | 4.3 参数：音乐节拍。可选值见 [Tempo 可选值](#tempo-options-v4-3)                                                                                                                                                           |
| Instrument        | String | 否   | 4.3 参数：音乐配器，最多支持 5 个。可选值见 [Instrument 可选值](#instrument-options-v4-3)                                                                                                                                   |
| Scene             | String | 否   | 4.3 参数：音乐场景，最多支持 3 个。可选值见 [Scene 可选值](#scene-options-v4-3)                                                                                                                                             |
| Lang              | String | 否   | 4.3 参数：指定语言，默认值 Chinese。可选值见 [Lang 可选值](#lang-options-v4-3)                                                                                                                                             |
| VodFormat         | String | 否   | 新增 v4.3 参数，音频格式，默认值为 wav。可选值："wav", "mp3"                                                                                                                                                                |

**ImplicitWaterMark 子字段说明：**

- **ContentProducer**（String）：生成合成服务提供者
- **ProduceId**（String）：内容制作编号
- **ContentPropagator**（String）：内容传播服务提供者
- **PropagateId**（String）：内容传播编号
- **Enable**（Bool）：启用开关
