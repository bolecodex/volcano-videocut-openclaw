DescribeSongJob 查询任务结果

返回结果示例：

{
    "Result": {
        "TaskID": "202408308513817850019840",
        "Status": 3,
        "Progress": 100,
        "FailureReason": {
            "Code": 300061,
            "Msg": "InputLyricsPlagiarized"
        },
        "SongDetail": {
            "AudioUrl": "xxxxx"
        }
    }
}

Status 任务状态说明：
0 -> 等待中
1 -> 处理中
2 -> 成功
3 -> 失败

FailureReason 字段说明：
任务失败原因，任务执行成功时为 null
- Code: 错误码
- Msg: 错误信息

SongDetail 字段说明：
- AudioUrl: 音频播放地址
