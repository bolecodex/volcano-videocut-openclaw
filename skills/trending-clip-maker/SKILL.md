# 热点内容批量生成 (Trending Clip Maker)

## 功能

利用 Seedance 2.0 批量生成多条短视频素材，用于社交媒体发布或投流素材中的插入片段。
支持 web_search 工具获取实时热点参考，实现最大化的 Seedance API 调用量。

## 内容主题

| 主题 | 说明 | 典型用途 |
|------|------|----------|
| `drama_highlight` | 戏剧性高光时刻 | 剧情类投流 |
| `visual_spectacle` | 视觉奇观大片 | 吸引眼球的开场 |
| `product_tease` | 产品预告片风格 | 带货/种草 |
| `emotion_hook` | 情感触动瞬间 | 情感类投流 |
| `action_energy` | 高能动作场面 | 动作/爽剧投流 |

## 使用方法

### 主题模式
选择一个预设主题，自动生成该主题下 N 条不同角度的视频。

### 自定义提示词模式
提供自己的提示词列表，每条提示词生成一段视频。

### Web搜索增强模式
启用 web_search 工具，Seedance 会搜索实时信息增强生成效果。

## 参数说明

- **内容主题**: 5种预设主题选择
- **自定义提示词**: 多条提示词，每条对应一个视频
- **生成数量**: 1-10条（默认4条）
- **每条时长**: 4-15秒（推荐5秒）
- **画面比例**: 9:16竖屏（推荐） / 16:9横屏 / 1:1方形
- **Web搜索**: 可选，使用热点搜索增强生成
- **搜索关键词**: Web搜索时的查询词

## 批量策略

推荐工作流：
1. 选择与剧情匹配的主题
2. 生成 4-8 条素材
3. 挑选最佳 2-3 条作为投流片头/插入片段
4. 与正片剪辑组合输出
