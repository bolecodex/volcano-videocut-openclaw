# 字幕处理

# 输入

- `transcription.json` （ASR 数据，需包含字幕文本`text`、逐字信息`words`（含单字`text`、起止时间`startTime/endTime`））
- `sensitivity.json` (敏感词字幕信息JSON文件)

# 输出

- `subtitles.ass` (ASS字幕文件，包含敏感词逐字替换后的字幕)

# 流程

- 1. 基于 `transcription.json` 和 `sensitivity.json` 生成 `subtitles.json`（不保存，包含敏感词逐字替换后的字幕）
- 2. 基于 `subtitles.json`生成 `subtitles.ass`（包含敏感词逐字替换后的字幕，字幕颜色为白色，字幕剧中，超出需要折行）
