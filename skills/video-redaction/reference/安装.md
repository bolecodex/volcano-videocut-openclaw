```

## 依赖清单

| 依赖   | 用途                | 安装方式                                    |
| ------ | ------------------- | ------------------------------------------- | --- |
| uv     | Python 依赖管理工具 | `curl -LsSf https://astral.sh/uv/install.sh | sh` |
| Demucs | 人声/背景音分离     | 通过 uv 自动安装（无需手动 pip）            |
| curl   | API 调用            | 系统自带                                    |

```
