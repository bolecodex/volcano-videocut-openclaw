# 违规画面打码 (Video Censor)

## 功能

调用 Ark 多模态模型分析视频关键帧，检测需要打码的违规内容（暴力、色情、敏感画面），并对指定时间范围应用马赛克/模糊。可选：结合字幕文件检测敏感字幕并打码。

## 使用方法

1. 在后期处理面板选择成片，点击「违规打码」。
2. 可选提供 SRT 字幕路径，用于敏感字幕检测。
3. 脚本会采样关键帧送审，输出打码后的视频。

## 前置条件

- `.env` 中配置 `ARK_API_KEY`。

## 输出

- 同目录生成 `*_censor.mp4`。
