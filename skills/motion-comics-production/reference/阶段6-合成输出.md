# 阶段 6：合成输出（拼接、合入音轨、烧录字幕）

目标：把所有片段拼成最终成片（可选合入音轨与字幕）。

## 0. 前置条件

- 已生成视频片段（阶段 5），确保每帧存在 `local_video_path`
- 如需配音：已生成音频（阶段 6A），确保对应帧存在 `local_audio_path`
- 如需字幕：已生成 `.yiwa/<project_id>/output/<project_id>.srt`（阶段 6B）

## 1. 合成

```bash
printf '%s' '{"project_id":"..."}' | node "scripts/compose-video.js"
```

输出：

```json
{ "success": true, "output_path": "...", "clips": 12 }
```

输出位置：

- `.yiwa/<project_id>/output/<project_id>.mp4`

## 2. 只合成部分片段（可选）

当用户只想预览某几段时，传 `clip_ids`：

```bash
printf '%s' '{"project_id":"...","clip_ids":["sb_001","sb_002"]}' | node "scripts/compose-video.js"
```

## 3. 字幕烧录的正确用法

- 脚本只有在发现 `.srt` 存在时才会烧录字幕。
- 推荐流程：先合成一次（用于音画对齐与写入 `merged_duration`）→ 生成 SRT → 再合成一次（烧录字幕并输出最终版本）。
