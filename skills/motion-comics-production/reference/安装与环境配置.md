# 安装与环境配置

所有命令都在 `motion-comics-production/` 目录执行（也就是 `SKILL.md` 所在目录），不要在 `scripts/` 目录里运行。

## 1. 环境要求

- Node.js 20+
- ffmpeg（需要 `ffmpeg` 与 `ffprobe` 在 PATH 中）

检查：

```bash
node --version
ffmpeg -version 2>&1 | head -1
ffprobe -version 2>&1 | head -1
```

## 2. 安装依赖

```bash
cd scripts
npm install
cd ..
```

## 3. 配置 .env

脚本会从 `motion-comics-production/.env` 读取环境变量（缺失时回退到系统环境变量）。

需要配置：

- `ARK_API_KEY`：火山方舟 API Key（图像/视频生成）  
  获取：https://console.volcengine.com/ark/region:ark+cn-beijing/apiKey

初始化：

```bash
cp .env.example .env
```

编辑 `.env` 填入真实值后，确认是否生效：

```bash
grep -E "^ARK_API_KEY=" .env
```

## 4. 验证 ARK_API_KEY

```bash
KEY=$(grep "^ARK_API_KEY=" .env | cut -d= -f2-)
curl -s -o /dev/null -w "%{http_code}" -H "Authorization: Bearer $KEY" https://ark.cn-beijing.volces.com/api/v3/models
```

返回 `200` 表示可用。


## 5. 安全约束

- 不要在对话中输出 `ARK_API_KEY` 明文；需要展示时只展示脱敏后片段。
- `.env` 仅用于本机/本仓库运行；不要提交到版本控制系统（如果仓库启用了 git）。
