# 阶段 5：视频片段生成

目标：把每帧分镜图转成 5 秒视频片段（本地 mp4 + 线上 URL），并回填到 `storyboards.json`。

## 0. 前置条件

- 该帧已经有分镜图（先跑阶段 4）
- `.env` 里 `ARK_API_KEY` 可用

## 1. 逐帧生成视频

```bash
printf '%s' '{"storyboard_id":"...", "project_id":"..."}' | node "scripts/generate-video.js"
```

可选：为某帧指定更明确的运动描述（覆盖默认描述）：

```bash
printf '%s' '{"storyboard_id":"...", "project_id":"...","motion_desc":"镜头推进，角色抬头微笑，风吹动衣角"}' | node "scripts/generate-video.js"
```

输出：

```json
{ "storyboard_id": "...", "video_url": "...", "local_path": "..." }
```

本地保存位置：

- `.yiwa/<project_id>/videos/<storyboard_id>.mp4`

## 2. 交互与重做

- 每生成一个片段都要告诉用户当前进度（已完成 X / 总 Y），并在阶段结束后询问是否需要替换某些片段。
- 替换某片段：重新跑该 `storyboard_id` 的命令即可（会覆盖本地文件与状态）。
