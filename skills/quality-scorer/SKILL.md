# 素材质量 AI 评分

使用 AI 自动评估投流素材的质量，预测效果，给出优化建议。

## 前置条件

- Python 3.10+，已安装 `openai` 和 `python-dotenv`
- `.env` 中配置 `ARK_API_KEY`
- FFmpeg 已安装（用于视频压缩）

## 工作流程

### 第一步：评分

```bash
# 评估最终成品
python3 scripts/score_quality.py video/output/promo_final.mp4

# 自定义输出目录
python3 scripts/score_quality.py video.mp4 -o scores/
```

### 第二步：查看报告

脚本输出 `score_{视频名}.json`，包含：

```json
{
  "overall_score": 85,
  "grade": "A",
  "scores": {
    "hook_score": 90,
    "rhythm_score": 85,
    "emotion_score": 80,
    "ending_score": 88,
    "dialogue_score": 82
  },
  "strengths": ["开头冲击力强", "节奏紧凑"],
  "weaknesses": ["中间有一处跳切"],
  "suggestions": ["建议在第15秒处增加转场"],
  "summary": "整体质量优秀，适合投放"
}
```

### 评分标准

| 等级 | 分数范围 | 说明 |
|------|---------|------|
| S 级 | 90+ | 顶级素材，可直接投放 |
| A 级 | 80-89 | 优秀素材，微调即可 |
| B 级 | 70-79 | 合格素材，建议优化 |
| C 级 | 60-69 | 需要较大调整 |
| D 级 | <60 | 建议重新剪辑 |

### 评分维度

1. **前3秒吸引力** -- 开头是否能让用户停住
2. **节奏流畅度** -- 片段衔接是否自然
3. **情绪曲线** -- 是否有清晰的情绪递进
4. **卡点效果** -- 结尾是否在最佳悬念位置
5. **台词完整性** -- 切点是否在台词自然停顿处

## 使用建议

- 低于 80 分的素材，根据 `suggestions` 字段进行优化
- 可以对比多个版本的评分，选择最高分版本投放
- 建议在完整流水线（分析→剪辑→字幕→BGM）后再评分
