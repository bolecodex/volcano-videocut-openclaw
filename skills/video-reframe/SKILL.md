# 横竖屏比例转换

基于 Seedance 2.0 的视频编辑能力，将现有视频智能转换为目标比例，无需手动裁切，由 AI 重新构图以适配竖屏或横屏。

## 功能

- **横屏转竖屏 (16:9 → 9:16)**：适合将横屏短剧/电影片段转为抖音、快手等竖屏投流素材
- **竖屏转横屏 (9:16 → 16:9)**：适合将竖屏素材转为横屏用于网页、TV 或信息流横版
- **转为方形 (1:1)**：可选 1:1 比例，适配部分信息流或封面

转换时保持原画面内容、人物与风格一致，由模型智能重构图，避免简单裁切导致的画面缺失。

## 使用场景

1. **投流素材适配**：横屏成片需投放抖音/快手，一键转竖屏
2. **多平台分发**：同一内容需同时提供横屏与竖屏版本
3. **历史素材复用**：竖屏素材需在横屏场景使用

## 参数说明

- **源视频**：待转换的视频文件（建议 2–15 秒、720p 以内、≤50MB）
- **目标比例**：`9:16` 竖屏、`16:9` 横屏、`1:1` 方形
- **输出时长**：4–15 秒（与源视频时长接近为佳）
- **自定义提示词**：可选，用于强调保留某类内容或构图偏好

## 命令行示例

```bash
# 横屏转竖屏（默认 9:16）
python scripts/seedance_reframe.py video/output/promo_01.mp4 -r 9:16 -o video/output

# 竖屏转横屏
python scripts/seedance_reframe.py video/output/clip.mp4 -r 16:9 -o video/output

# 快速模型、自定义输出名
python scripts/seedance_reframe.py video/source.mp4 -r 9:16 -n douyin_ver --fast
```

## 前置条件

- `.env` 中配置 `ARK_API_KEY`（火山引擎方舟）
- 已安装 FFmpeg（用于源视频预处理与时长检测）
- 源视频格式为 mp4，建议分辨率 720p 及以下、单段 2–15 秒

## 输出

在指定输出目录下生成 `reframe_<源文件名>_<比例>.mp4`，或通过 `-n` 指定文件名。
