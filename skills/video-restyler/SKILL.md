# 视频风格化编辑 (Video Restyler)

## 功能

利用 Seedance 2.0 的视频编辑能力，在保持原视频动作和构图的基础上进行风格化改造：
- 更换场景环境（室内→室外、现代→古代）
- 改变天气/光照（晴天→雨天、白天→夜晚）
- 添加视觉特效（雪花、霓虹、粒子效果）
- 风格迁移（写实→赛博朋克、现代→古风）

## 风格预设

| 预设 | 说明 | 适用场景 |
|------|------|----------|
| `night_scene` | 转换为月光夜景 | 增加氛围感 |
| `snow_effect` | 添加雪景效果 | 冬季/唯美场景 |
| `rain_mood` | 添加雨景+霓虹反射 | 情感/都市剧 |
| `golden_hour` | 金色黄昏光线 | 温馨/浪漫场景 |
| `luxury_upgrade` | 升级为高端质感 | 豪门/商战剧 |
| `cyberpunk` | 赛博朋克风格 | 科幻/未来感 |
| `ancient_chinese` | 中国古风化 | 古装/仙侠剧 |
| `horror_tint` | 恐怖氛围渲染 | 悬疑/惊悚剧 |

## 参数说明

- **源视频**: 需要风格化的视频片段（15秒以内）
- **风格预设**: 8种预设风格选择
- **自定义编辑指令**: 描述想要的编辑效果
- **参考图片**: 可选，提供目标风格的参考图
- **时长**: 4-15秒
- **画面比例**: 自适应 / 16:9 / 9:16 / 1:1

## 典型用途

1. 为投流素材增加视觉冲击力
2. 同一片段生成多种风格版本用于A/B测试
3. 将普通画面升级为电影质感
4. 适配不同类型剧情的视觉基调
