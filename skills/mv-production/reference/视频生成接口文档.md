# Seedance 1.5 Pro 视频生成接口文档

## 接口地址

```
POST https://ark.cn-beijing.volces.com/api/v3/contents/generations/tasks
```

## 鉴权方式

本接口仅支持 API Key 鉴权，请在获取 API Key 页面，获取长效 API Key。

---

## 支持的生成能力

Seedance 1.5 Pro 支持以下视频生成能力：

1. **文生视频**：根据文本提示词生成视频
2. **图生视频-首帧**：根据首帧图片 + 文本提示词生成视频
3. **图生视频-首尾帧**：根据首帧图片 + 尾帧图片 + 文本提示词生成视频
4. **有声视频**：支持自定义是否包含音频
5. **样片模式(draft)**：快速生成预览视频
6. **基于样片生成正式视频**：基于 Draft 视频生成高质量正式视频

## 请求参数

### 请求体参数

| 参数名  | 类型     | 必填 | 说明                                       |
| ------- | -------- | ---- | ------------------------------------------ |
| model   | string   | 是   | 模型名称：`doubao-seedance-1-5-pro-251215` |
| content | object[] | 是   | 输入给模型的内容，支持文本、图片、视频格式 |

### content 内部参数

#### 文本信息

| 参数名 | 类型   | 必填 | 说明                                                                              |
| ------ | ------ | ---- | --------------------------------------------------------------------------------- |
| type   | string | 是   | 内容类型，固定值：`text`                                                          |
| text   | string | 是   | 文本提示词，描述期望生成的视频。支持中英文，建议中文不超过500字，英文不超过1000词 |

#### 图片信息

| 参数名        | 类型   | 必填     | 说明                                                                           |
| ------------- | ------ | -------- | ------------------------------------------------------------------------------ |
| type          | string | 是       | 内容类型，固定值：`image_url`                                                  |
| image_url     | object | 是       | 图片对象                                                                       |
| image_url.url | string | 是       | 图片URL或Base64编码。URL需可访问；Base64格式：`data:image/png;base64,{base64}` |
| role          | string | 条件必填 | 图片位置/用途：`first_frame`(首帧)、`last_frame`(尾帧)                         |

**图片要求：**

- 格式：jpeg、png、webp、bmp、tiff、gif、heic、heif
- 宽高比：(0.4, 2.5)
- 宽高长度：300-6000px
- 大小：小于30MB

#### 样片信息 (仅1.5 Pro支持)

| 参数名        | 类型   | 必填 | 说明                             |
| ------------- | ------ | ---- | -------------------------------- |
| type          | string | 是   | 内容类型，固定值：`draft_task`   |
| draft_task    | object | 是   | 样片任务对象                     |
| draft_task.id | string | 是   | 样片任务ID，基于此ID生成正式视频 |

### 生成参数

| 参数名                  | 类型    | 默认值   | 说明                                                            |
| ----------------------- | ------- | -------- | --------------------------------------------------------------- |
| callback_url            | string  | -        | 回调通知地址，任务状态变化时推送POST请求                        |
| return_last_frame       | boolean | false    | 是否返回生成视频的尾帧图像                                      |
| service_tier            | string  | default  | 服务等级：`default`(在线推理)、`flex`(离线推理，价格为50%)      |
| execution_expires_after | integer | 172800   | 任务超时时间(秒)，取值范围[3600, 259200]，默认48小时            |
| generate_audio          | boolean | true     | **仅1.5 Pro支持**，是否生成同步声音                             |
| draft                   | boolean | false    | **仅1.5 Pro支持**，是否开启样片模式                             |
| resolution              | string  | 720p     | 视频分辨率：`480p`、`720p`、`1080p`                             |
| ratio                   | string  | adaptive | 宽高比：`16:9`、`4:3`、`1:1`、`3:4`、`9:16`、`21:9`、`adaptive` |
| duration                | integer | 5        | 视频时长(秒)，支持2-12秒；设为-1则由模型自主选择                |
| seed                    | integer | -1       | 种子整数，取值[-1, 2^32-1]，-1为随机                            |
| camera_fixed            | boolean | false    | 是否固定摄像头                                                  |
| watermark               | boolean | false    | 是否包含水印                                                    |

## 请求示例

### 1. 文生视频

```json
{
  "model": "doubao-seedance-1-5-pro-251215",
  "content": [
    {
      "type": "text",
      "text": "小猫对着镜头打哈欠"
    }
  ],
  "resolution": "720p",
  "ratio": "16:9",
  "duration": 5,
  "seed": 11,
  "camera_fixed": false,
  "watermark": true,
  "generate_audio": true
}
```

### 2. 图生视频-首帧

```json
{
  "model": "doubao-seedance-1-5-pro-251215",
  "content": [
    {
      "type": "image_url",
      "image_url": {
        "url": "https://example.com/image.jpg"
      },
      "role": "first_frame"
    },
    {
      "type": "text",
      "text": "小猫对着镜头打哈欠，然后伸懒腰"
    }
  ],
  "resolution": "720p",
  "ratio": "16:9",
  "duration": 5,
  "generate_audio": true
}
```

### 3. 图生视频-首尾帧

```json
{
  "model": "doubao-seedance-1-5-pro-251215",
  "content": [
    {
      "type": "image_url",
      "image_url": {
        "url": "https://example.com/first_frame.jpg"
      },
      "role": "first_frame"
    },
    {
      "type": "image_url",
      "image_url": {
        "url": "https://example.com/last_frame.jpg"
      },
      "role": "last_frame"
    },
    {
      "type": "text",
      "text": "花朵缓缓绽放的过程"
    }
  ],
  "resolution": "720p",
  "ratio": "16:9",
  "duration": 5,
  "generate_audio": true
}
```

### 4. 生成无声视频

```json
{
  "model": "doubao-seedance-1-5-pro-251215",
  "content": [
    {
      "type": "text",
      "text": "小猫对着镜头打哈欠"
    }
  ],
  "generate_audio": false,
  "duration": 5
}
```

### 5. 样片模式 (Draft)

```json
{
  "model": "doubao-seedance-1-5-pro-251215",
  "content": [
    {
      "type": "text",
      "text": "小猫对着镜头打哈欠"
    }
  ],
  "draft": true,
  "resolution": "480p"
}
```

### 6. 基于样片生成正式视频

```json
{
  "model": "doubao-seedance-1-5-pro-251215",
  "content": [
    {
      "type": "draft_task",
      "draft_task": {
        "id": "draft-task-id-xxx"
      }
    }
  ],
  "watermark": false,
  "generate_audio": true
}
```

### 7. 返回尾帧图像

```json
{
  "model": "doubao-seedance-1-5-pro-251215",
  "content": [
    {
      "type": "text",
      "text": "小猫对着镜头打哈欠"
    }
  ],
  "return_last_frame": true,
  "duration": 5
}
```

---

## 响应参数

| 参数名 | 类型   | 说明                      |
| ------ | ------ | ------------------------- |
| id     | string | 视频生成任务ID，仅保存7天 |

**说明：** 创建视频生成任务为异步接口，获取ID后，需要通过查询视频生成任务API来查询任务状态。任务成功后，会输出生成视频的video_url。

---

## 回调状态说明

回调返回的status包括：

- `queued`：排队中
- `running`：任务运行中
- `succeeded`：任务成功
- `failed`：任务失败
- `expired`：任务超时
