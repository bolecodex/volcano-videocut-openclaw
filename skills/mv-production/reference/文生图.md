# Seedream 5.0 图像生成接口文档

## 接口地址

```
POST https://ark.cn-beijing.volces.com/api/v3/images/generations
```

## 鉴权方式

本接口仅支持 API Key 鉴权，请在获取 API Key 页面，获取长效 API Key。

## 支持的生成能力

Seedream 5.0 支持以下图像生成能力：

1. **文生图**：纯文本输入生成单张图片
2. **图生图**：单图输入进行图像编辑
3. **多图融合**：多张参考图融合生成新图像
4. **组图生成**：单图/多图生成多张内容关联的图片
5. **联网搜索**：融合实时网络信息生成图片
6. **流式输出**：实时返回生成进度

---

## 请求参数

### 公共参数

| 参数名                              | 类型            | 必填 | 说明                                                  |
| ----------------------------------- | --------------- | ---- | ----------------------------------------------------- |
| model                               | string          | 是   | 模型ID：`doubao-seedream-5-0-260128`                  |
| prompt                              | string          | 是   | 文本提示词，建议不超过300个汉字或600个英文单词        |
| size                                | string          | 是   | 输出图像尺寸，支持 `2K`、`3K` 或像素值如 `2048x2048`  |
| output_format                       | string          | 否   | 输出格式：`png`、`jpeg`，默认 `png`                   |
| response_format                     | string          | 否   | 返回格式：`url`(链接)、`b64_json`(Base64)，默认 `url` |
| watermark                           | boolean         | 否   | 是否添加水印，默认 `false`                            |
| stream                              | boolean         | 否   | 是否开启流式输出，默认 `false`                        |
| image                               | string/string[] | 否   | 参考图片URL或Base64编码                               |
| sequential_image_generation         | string          | 否   | 组图开关：`disabled`(关闭)、`auto`(开启)              |
| sequential_image_generation_options | object          | 否   | 组图选项                                              |
| optimize_prompt_options             | object          | 否   | 提示词优化选项                                        |
| tools                               | object[]        | 否   | 工具配置，如联网搜索                                  |

### 组图选项 (sequential_image_generation_options)

| 参数名     | 类型   | 说明         |
| ---------- | ------ | ------------ |
| max_images | number | 最大图片数量 |

### 提示词优化选项 (optimize_prompt_options)

| 参数名 | 类型   | 说明                                       |
| ------ | ------ | ------------------------------------------ |
| mode   | string | 优化模式：`standard`(高质量)、`fast`(快速) |

### 工具配置 (tools)

| 参数名 | 类型   | 说明                             |
| ------ | ------ | -------------------------------- |
| type   | string | 工具类型：`web_search`(联网搜索) |

---

## 图片输入要求

- 格式：jpeg、png、webp、bmp、tiff、gif
- 宽高比范围：[1/16, 16]
- 宽高长度：> 14px
- 大小：不超过 10 MB
- 总像素：不超过 6000×6000 = 36000000 px
- 最多支持传入 14 张参考图

---

## 响应参数

| 参数名      | 类型   | 说明             |
| ----------- | ------ | ---------------- |
| data        | array  | 生成图片数据列表 |
| data[].url  | string | 图片下载链接     |
| data[].size | string | 图片尺寸         |
| usage       | object | 使用量信息       |

## 请求示例

### 一、纯文生图（文生图）

#### 1.1 基础文生图

```json
{
  "model": "doubao-seedream-5-0-260128",
  "prompt": "充满活力的特写编辑肖像，模特眼神犀利，头戴雕塑感帽子，色彩拼接丰富，眼部焦点锐利，景深较浅，具有Vogue杂志封面的美学风格，采用中画幅拍摄，工作室灯光效果强烈。",
  "size": "2K",
  "output_format": "png",
  "response_format": "url",
  "watermark": false
}
```

#### 1.2 联网搜索文生图

```json
{
  "model": "doubao-seedream-5-0-260128",
  "prompt": "制作一张上海未来5日的天气预报图，采用现代扁平化插画风格，清晰展示每日天气、温度和穿搭建议。整体为横向排版，标题为"上海未来5日天气预报"，包含5个等宽的垂直卡片，从左到右依次排列。整体风格为现代、干净、友好的扁平化矢量插画风格，线条清晰，色彩柔和。人物形象采用年轻男女的卡通插画，表情自然，姿态放松，服装细节清晰。",
  "size": "2048x2048",
  "output_format": "png",
  "response_format": "url",
  "tools": [
    {
      "type": "web_search"
    }
  ],
  "watermark": false
}
```

#### 1.3 流式输出文生图

```json
{
  "model": "doubao-seedream-5-0-260128",
  "prompt": "参考图1，生成四图片，图中人物分别带着墨镜，骑着摩托，带着帽子，拿着棒棒糖",
  "image": "https://example.com/reference.jpg",
  "size": "2K",
  "sequential_image_generation": "auto",
  "sequential_image_generation_options": {
    "max_images": 4
  },
  "output_format": "png",
  "response_format": "url",
  "stream": true,
  "watermark": false
}
```

### 二、图文生成（图生图）

#### 2.1 单图生单图（图生图）

基于已有图片，结合文字指令进行图像编辑。

```json
{
  "model": "doubao-seedream-5-0-260128",
  "prompt": "保持模特姿势和液态服装的流动形状不变。将服装材质从银色金属改为完全透明的清水（或玻璃）。透过液态水流，可以看到模特的皮肤细节。光影从反射变为折射。",
  "image": "https://ark-project.tos-cn-beijing.volces.com/doc_image/seedream4_5_imageToimage.png",
  "size": "2K",
  "output_format": "png",
  "response_format": "url",
  "watermark": false
}
```

#### 2.2 多图融合（多图生单图）

根据文本描述和多张参考图片，融合它们的风格、元素等特征来生成新图像。

```json
{
  "model": "doubao-seedream-5-0-260128",
  "prompt": "将图1的服装换为图2的服装",
  "image": [
    "https://ark-project.tos-cn-beijing.volces.com/doc_image/seedream4_imagesToimage_1.png",
    "https://ark-project.tos-cn-beijing.volces.com/doc_image/seedream4_5_imagesToimage_2.png"
  ],
  "size": "2K",
  "sequential_image_generation": "disabled",
  "output_format": "png",
  "response_format": "url",
  "watermark": false
}
```

## 提示词建议

建议用简洁连贯的自然语言写明 **主体 + 行为 + 环境**，若对画面美学有要求，可用自然语言或短语补充 **风格、色彩、光影、构图** 等美学元素。

- 文本提示词建议不超过 **300个汉字** 或 **600个英文单词**
- 字数过多信息容易分散，模型可能因此忽略细节

---

## 使用限制

- **任务数据保存时间**：仅保留 **24小时**，超时后自动清除
- **RPM 限流**：账号下同模型每分钟生成图片数量有上限
- **图片输入限制**：
  - 格式：jpeg、png、webp、bmp、tiff、gif
  - 宽高比：[1/16, 16]
  - 大小：≤ 10 MB
  - 总像素：≤ 36000000 px
  - 最多 14 张参考图

---

## 流式输出事件类型

| 事件类型                           | 说明             |
| ---------------------------------- | ---------------- |
| image_generation.partial_failed    | 部分图片生成失败 |
| image_generation.partial_succeeded | 部分图片生成成功 |
| image_generation.completed         | 全部完成         |
| image_generation.partial_image     | 部分图像数据     |
