## 脚本说明

`voice_segment.py` 会根据 `asr_split.json` 中的 `segments` 时间轴，把人声音频（通常是 `vocals.mp3`）切分成多个片段，并输出映射文件。

- 脚本路径：`scripts/base/voice_segment.py`
- 典型输入：
  - 人声音频：`vocals.mp3`
  - ASR 切分结果：`asr_split.json`
- 典型输出：
  - `segments/segment_0001.wav`、`segments/segment_0002.wav`...
  - `segments/mapping.json`

## 使用方式（uv）

在 `scripts/base` 目录执行：

```bash
uv run voice_segment.py \
  "/path/to/vocals.mp3" \
  "/path/to/asr_split.json" \
  -o "/path/to/output/video_name"
```

## 参数说明

| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| `voice_audio`（位置参数） | 人声音频路径（如 `vocals.mp3`） | 必填 |
| `asr_split_json`（位置参数） | ASR 切分结果 JSON 路径（如 `asr_split.json`） | 必填 |
| `-o`, `--output-dir` | 输出目录（例如 `output/<剧名>/<视频名>/`） | 必填 |
| `--format` | 切片格式：`wav` 或 `mp3` | `wav` |
| `--segments-dirname` | 片段子目录名 | `segments` |
| `--pad-start` | 每个片段开始前补偿秒数 | `0.0` |
| `--pad-end` | 每个片段结束后补偿秒数 | `0.0` |
| `--min-duration` | 最小时长阈值（低于阈值会跳过） | `0.0` |

## 输出文件说明

`mapping.json` 结构示例：

```json
[
  {
    "id": 1,
    "source_id": 13,
    "start": 12.34,
    "end": 15.67,
    "duration": 3.33,
    "text": "示例文本",
    "file": "segment_0001.wav",
    "speaker": "spk_0"
  }
]
```

字段说明：

- `id`：本次切片后的连续编号
- `source_id`：来源于 ASR `segments` 的原始 id
- `start/end/duration`：切片时间信息（秒）
- `text`：对应识别文本
- `file`：生成的片段文件名
- `speaker`：可选，说话人信息（ASR 中有则保留）
