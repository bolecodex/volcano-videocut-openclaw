# 输入&输出

## 流程总览（按 SKILL）

> 以现有脚本与文档为准，按阶段串联执行。

1. `环境配置.md`：准备 Python/uv、base 环境、index-tts 隔离环境
2. `音视频预处理.md`：视频静音 + 音轨提取 + 人声/背景音分离 + ASR 原始结果
3. `ASR结果与处理.md`：将 `asr.json` 处理为 `asr_split.json`
4. `人声片段剪切.md`：基于 `asr_split.json` 切分 `vocals` 生成 `segments/`
5. `人声语速预估.md`：对片段做语速估计，输出 `speed_profile/`
6. `字幕翻译.md`：结合 `asr_split` 与语速进行本地化翻译，输出 `transcript_{lang}.json`
7. `TTS音色复刻.md`：基于翻译结果 + 片段音频做复刻，输出 `tts/{lang}/`
8. `音视频合并.md`：按 `asr_split` 时间轴合并 TTS + 背景音并与静音视频合成
9. `字幕压制.md`：将字幕压制到合成视频，输出最终成片

## 输入要求

- 输入视频支持：文件路径、https 链接、视频文件夹 三种方式
  - 文件路径：例如 `/path/to/video.mp4`
  - https 链接：例如 `https://example.com/video.mp4`
  - 视频文件夹：例如 `/path/to/video_folder/`， 支持递归处理子文件夹下的视频文件
- `video_asr_pipeline.py` 支持批量输入多个参数（多个文件路径 / 多个 https 链接 / 混合）
- 批量输入时最多只允许一个文件夹路径，超过会报错
- 输出文件夹默认为 `../../../output/`（相对于 `video-translation` 根目录），支持用户自定义

## 输出要求

- 输出文件夹默认为 `../../../output/`（相对于 `video-translation` 根目录），支持用户自定义
- 每个剧集对应一个输出文件夹，输出文件夹名为剧集名称（不包含扩展名）
- 用户未指定剧集时，默认输出到 `../../../output/` 下
- 按照剧集 `../../../output/${剧名}/${视频文件名}` 下的所有视频文件，每个视频文件对应一个输出文件夹，输出文件夹名为视频文件名（不包含扩展名）

### 当前 `video_asr_pipeline.py` 实际输出

- 每个输出文件夹下，当前脚本会输出以下文件：
  - 分离后的音频文件（`audio.mp3`）
  - 原始视频文件（`original_video.mp4`）
  - 静音视频文件（`muted_video.mp4`）
  - 切分后的人声文件（`vocals.mp3`）
  - 背景音文件（`background.mp3`）
  - asr 原始结果（`asr.json`）
  - asr 切分结果（`asr_split.json`，由 ASR 结果处理阶段产出）
  - 单任务状态文件（`status.json`）
- 剧集目录下会输出批量汇总（`pipeline_report.json`）

### 其他产物说明

- 以下产物属于后续翻译/TTS/字幕合成链路，当前 `video_asr_pipeline.py` 不生成：
  - `video_without_subtitles.mp4`
  - `replicated.mp3`
  - `translates/transcript_zh.json`、`translates/transcript_en.json`
  - `subtitles_generated.srt`
  - `tts/segment_tts_0001.mp3` 等
  - `segments/segment_0001.mp3` 等
  - `${视频文件名}_translate_video.mp4`
  - `localized_name.json`
