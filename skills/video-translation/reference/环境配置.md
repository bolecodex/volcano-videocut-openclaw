## python & uv 检测与安装

### 环境要求

- Python 3.12+
- UV

### 环境检测

在安装之前，建议先检测系统中是否已存在 Python 和 UV。

#### Linux/macOS:

```bash
# 检测 Python 版本
python3 --version

# 检测 UV 是否安装
uv --version
```

#### Windows:

```powershell
# 检测 Python 版本
python --version

# 检测 UV 是否安装
uv --version
```

### 安装步骤

#### Linux/macOS:

```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

#### Windows:

```
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

## index-TTS 环境检测与安装

```bash
# 在 video-translation 根目录执行
bash scripts/setup_index_tts_env.sh
```

上面的脚本会自动完成：

- 准备/拉取 `scripts/vendors/index-tts`
- `uv sync --all-extras` 安装 index-tts 环境
- 检查并下载 IndexTTS checkpoints（缺失时自动下载）
- 校验关键模型文件是否齐全
- 回到 `scripts/base` 执行 `uv sync --dev`

如果只更新主流程代码而没有改动 TTS，可直接在 `scripts/base` 执行：

```bash
uv sync --dev
```

如网络访问 HuggingFace 较慢，可先设置镜像再执行脚本：

```bash
export HF_ENDPOINT="https://hf-mirror.com"
bash scripts/setup_index_tts_env.sh
```
