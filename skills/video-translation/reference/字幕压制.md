# 字幕压制

## 流程阶段

- 阶段 8（成片输出）
- 上游：`字幕翻译.md`、`音视频合并.md`
- 下游：最终成片交付

## 目标

将翻译结果字幕压制到视频中，支持默认“剧中、靠下”样式，并允许用户完整自定义。

## 规则

1. 使用 `ffmpeg/ffprobe` 读取视频宽高，根据分辨率自动计算默认字号与底部安全边距。
2. 默认位置：**水平居中、靠下安全区**（默认底部 10%，左右各 5%）。
3. 超长字幕自动折行；默认不省略文本（不再追加 `…`）。
4. 默认样式：
   - 字体颜色：白色（`#FFFFFF`）
   - 阴影颜色：黑色（`#000000`）
   - 描边颜色：黑色（`#000000`）
5. 以上参数均支持用户自定义。

## 脚本

- `scripts/base/burn_subtitle.py`

## 输入

- `--video-input`：输入视频（通常为音视频合并结果）
- `--transcript-json`：翻译结果 JSON（需包含 `segments[].start/end/text`）

## 输出

- `--output-video`：压制字幕后视频
- `--ass-output`（可选）：导出 ASS 字幕文件，便于手工检查/二次编辑

## 参数说明

| 参数 | 说明 | 默认值 |
| --- | --- | --- |
| `--video-input` | 输入视频路径 | 必填 |
| `--transcript-json` | 翻译结果 JSON 路径 | 必填 |
| `--output-video` | 输出视频路径 | 必填 |
| `--ass-output` | 导出 ASS 路径（可选） | 无（临时 ASS） |
| `--font-name` | 字体名称 | `Arial` |
| `--font-size` | 字号（`0` 表示自动） | `0`（自动） |
| `--font-color` | 字体颜色 | `#FFFFFF` |
| `--outline-color` | 描边颜色 | `#000000` |
| `--shadow-color` | 阴影颜色 | `#000000` |
| `--outline` | 描边宽度 | `1.5` |
| `--shadow` | 阴影大小 | `2.0` |
| `--margin-l` | 左侧安全边距像素（`0` 自动） | `0`（自动，约视频宽 5%） |
| `--margin-r` | 右侧安全边距像素（`0` 自动） | `0`（自动，约视频宽 5%） |
| `--margin-v` | 底部安全边距像素（`0` 自动） | `0`（自动） |
| `--max-lines` | 最大行数 | `2` |
| `--max-width-ratio` | 字幕最大宽度占视频宽比例 | `0.90` |
| `--retry-times` | 压制失败重试次数 | `3` |

## 使用示例（uv）

```bash
uv run burn_subtitle.py \
  --video-input "/path/to/output/剧名/视频名/视频名_translate_video.mp4" \
  --transcript-json "/path/to/output/剧名/视频名/translates/transcript_en.json" \
  --output-video "/path/to/output/剧名/视频名/视频名_translate_video_subbed.mp4"
```

## 默认布局口径（自动）

- 字号：约短边的 4.5%（最小 24）
- 位置：底部居中（ASS Alignment=2）
- 底部边距：约视频高度的 10%（最小 24）
- 左右边距：约视频宽度的 5%（最小 24）

## 日志

- 日志前缀：`[burn_subtitle]`
- 输出关键信息：分辨率、片段数、字号、边距、输出路径、重试信息
