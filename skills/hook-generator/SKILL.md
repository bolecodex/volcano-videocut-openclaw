# AI开场钩子生成 (Hook Generator)

## 功能

利用 Seedance 2.0 的文生视频和图生视频能力，生成高冲击力的 5-8 秒开场钩子视频片段。
遵循"三秒法则"——在视频最开始通过超现实、高戏剧性的 AI 画面抓住观众注意力。

生成的钩子视频可以直接拼接在正片/投流素材前面，大幅提升完播率和点击率。

## 钩子风格预设

| 风格 | 说明 | 适用场景 |
|------|------|----------|
| `suspense_zoom` | 悬疑慢速推进 | 悬疑/惊悚剧 |
| `explosion_reveal` | 粒子爆炸揭示 | 动作/爽剧 |
| `emotional_rain` | 情感雨景特写 | 情感/虐恋剧 |
| `epic_slow_motion` | 史诗慢镜头 | 古装/仙侠剧 |
| `glitch_rewind` | 故障回放效果 | 重生/穿越剧 |
| `luxury_reveal` | 奢华揭幕效果 | 豪门/都市剧 |
| `mystery_approach` | 神秘逼近镜头 | 悬疑/恐怖剧 |

## 使用方法

### 基础：纯文字生成
选择一个钩子风格，自动生成开场钩子。

### 进阶：图生视频
提供剧中的一个高冲击力画面作为首帧，AI 将基于该画面生成动态钩子。

### 高级：从源视频提取
提供源视频，工具自动提取关键帧并生成匹配风格的钩子。

## 参数说明
- **钩子风格**: 7种预设风格选择
- **自定义提示词**: 补充描述想要的画面效果
- **首帧图片**: 可选，指定生成视频的第一帧
- **源视频**: 可选，自动提取关键帧
- **时长**: 4-10秒（推荐5秒）
- **画面比例**: 9:16竖屏 / 16:9横屏 / 1:1
